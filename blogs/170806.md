
# 金华卷                  一片泪水

  涛声依旧。一个浪从海里奔出，跃起，在阳光下闪耀了一瞬，接着坠在滩上，摔得粉身碎骨。一颗颗水珠从粉碎中溅起，哗啦啦的声音，到蚂蚁的耳里化为一声声惨叫。

  凝望着大海，看着水天相接的一线，闪闪发光，那是一条银线。“我一生也到不了那了。”突然，那条线绷直，飞向蚂蚁，紧紧狠狠地勒住了蚂蚁的脖子。

  蚂蚁蚂蚁，躺在沙滩上。细小的沙子是一堆乱石，蚂蚁望着四周深深感到自己是如此的渺小。“广远的世界……大海的尽头……”蚂蚁闭着眼睛，开始一动不动，仿佛成了沙子中的一小粒，最不起眼的一粒。可它连一粒沙子也当不成。阳光，一束束针，扎入蚂蚁的外骨，抽取它的水分，渐渐把它变成一具枯壳。蚂蚁不动，悲伤地让灿烂的太阳审判它吧。

  突然，一个浪扑向了它，蚂蚁顿时被抱入了海中。GUGU，是气泡翻滚的声音，惊碎了蚂蚁的悲伤梦，化为泡沫。睁开眼睛，看，这是大海里。

  蚂蚁拼命地挣扎，六足伸曲甩动，可海水是如此的强壮坚固，一股股力量从四面八方涌来，不可抗拒。涌流撞击在一起，交融在一起。一个浪升腾起来，耀眼，，可一会儿便塌下了。另个浪升起，扑向天空。

  海的声音震动蚂蚁的耳朵，穿透蚂蚁的身躯。翻滚，随着流，游动，旋转，升起，攀着浪，蚂蚁觉得自己变成了一滴水，与大海同呼吸，与周围的水有同样的心跳……可它做不了一滴水，水闷住它的呼吸，挤压它的外壳。最终，它被甩到了岸上，伴随它的，还有许多水珠。

  蚂蚁看着眼前壮阔的，明白大海是由千千万万滴水组成的，奔腾，开拓，是它们的使命。自己不是沙子，也不是水滴，而是一只活生生的蚂蚁。自己要做的，自己的意义，不是去大海的尽头，而是与自己的团体一起奋斗。

大海的尽头也许很美，也许很荒凉。但蚂蚁不再留念这大海了，这不再有意义，自己这儿，也是大海的尽头。它要投入自己的团体中去。

只留下一滴感慨的泪水，滴入大海中，随着海流到那一边尽头，蚂蚁的幻想。

# 本来我可以不急

_小店从旁边搬到了遥远的一边，极目看去，小店的踪影是一个上有“烟酒副食”的牌子，孤零零地立着。_

  _喘息声如海浪上上下下，匆忙的脚步把弟弟甩在后头。遥远的距离，光秃秃的地面，刺眼的太阳，烂塌的小草，热腾腾的风烦厌地踢了褪色垃圾一脚，滚啊滚，急躁的我把脚步向前投去，奔阿奔。_

  _小店的牌子还是那么远，再眼中变大得那么慢，像老人一样缓缓__……_

  _皱巴巴的皮架在骨头上，就成了僵硬的脸。年轻的肌肉操纵着脸皮，晒出不耐烦的表情。老人接过我的钱，把头转向左边的桌子。他像个老树，在轮椅上，扎了根。他头动得那么慢，看起来吃力，好似我用眼睛拧动他的。然后他伸出右手，朝着抽屉——右边上数下第二个抽屉，我早就清清楚楚。_

  _好慢，好慢，好慢！我耐不住，手一动，飞快地去拉那放钱的抽屉。场景渐渐变模糊了，我的手，变成了罪恶的匕首。“啪！”_

  _多么真实，多么响亮，我的脚被来自回忆的一股寒冻住了，不迈步。_

  _老头猛地拍了下去，用了他全部力气。责备的目光直直射向我。“那么急！”_

  _店里顿时寂静了。_

_抽屉被拉开，老头拿出一叠老旧但整齐的纸币，抽出，将找回的钱递给我。那堆纸币乱了。我目光转回来，探出右手接。这时发现手臂上的掌印是多么红，正如我的脸。。_

  _想到这里，我不禁感到尴尬。本来我等一下就好了，结果因为心急惹得老人生气。不过，老人也用不着这样这样做啊。_

  _想着想着，我放慢了脚步，从容地走路，平息下呼吸。不急，看牌子在慢慢靠近，等待着到达。_
